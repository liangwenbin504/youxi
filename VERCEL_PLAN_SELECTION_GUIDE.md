# Vercel 计划选择指南

## 正确选择：个人项目（Hobby 计划）

你现在看到的是 Vercel 的计划选择页面，**请选择 `I'm working on personal projects`（我正在处理个人项目）**，然后点击 `Continue` 按钮。

### 选择原因：
1. **你的项目是个人项目**：`youxi` 仓库看起来是你的个人网站/游戏项目
2. **Hobby 计划是免费的**：适合个人开发者使用
3. **避免不必要的 Pro 计划试用**：Pro 计划有14天试用期，到期后可能会产生费用
4. **Hobby 计划功能足够**：提供免费的静态网站部署、自动部署、HTTPS 等核心功能

### 操作步骤：
1. 点击 `I'm working on personal projects` 选项（单选按钮）
2. 在 `Team Name` 输入框中，**输入你的名字或项目名称**（例如：`liangwenbin` 或 `youxi`）
3. 点击 `Continue` 按钮继续
4. 接下来按照提示连接你的 GitHub 账号，然后选择 `youxi` 仓库进行部署

这样选择后，你就可以使用 Vercel 的免费计划部署你的静态网站了。
